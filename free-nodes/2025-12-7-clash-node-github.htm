第一章 认识这只"电子老鼠"

1.1 工具本质解码

老鼠台本质上是一个智能流量中继系统，采用Socks5代理协议与AES-256加密算法的组合架构。其核心技术在于动态端口跳转技术（DPT），使得防火墙难以通过特征检测识别流量。与传统VPN的全流量隧道不同，它采用智能分流机制，仅对受限网站请求进行代理，这种设计使其速度比常规方案提升40%以上。

1.2 三大核心优势

  • 隐形的数字斗篷：通过模仿HTTPS流量特征，使得深度包检测（DPI）系统将其误判为普通加密网页流量
  • 硬件级效能：实测在树莓派4B上能维持20Mbps的稳定传输，内存占用始终低于35MB

FAQ

Vmess 与 Trojan 哪个更快？
在相同环境下，Trojan 通常连接更快，因为它使用原生 TLS 通道；Vmess 则稍有额外的协议开销。但在配置合理的情况下，两者的速度差距不大，主要取决于服务器线路质量。
QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
Clash for Android 如何导入订阅并自动更新？
在 Clash for Android 的配置界面中选择“配置文件”→“新建配置”，粘贴订阅链接即可。启用自动更新功能后，软件会定期检查订阅源是否有新节点，并在后台自动替换旧配置，保持节点最新状态。
Potatso Lite 支持订阅链接导入吗？
Potatso Lite 不支持自动订阅功能，用户需手动添加节点。如果想批量导入订阅，可使用其他客户端生成节点二维码，再手动扫描添加。适合轻量使用者。
Nekoray 如何导入 V2Ray 节点？
打开 Nekoray 客户端后，点击“导入配置”，选择 V2Ray 或 Clash 类型的订阅链接。程序会自动解析节点列表，用户可一键连接或进行延迟测试。
Trojan-Go 与 Trojan 有何不同？
Trojan-Go 是 Trojan 的分支版本，支持多路复用、QUIC、WebSocket 等新特性。它在兼容性和性能上都有显著提升，适合与现代客户端（如 Clash Meta、Sing-box）搭配使用。
Nekoray 的节点导入是否支持批量 URL？
支持。Nekoray 可通过订阅 URL 批量导入节点，并定期更新。用户无需手动添加每个节点，保证节点列表始终最新，提高使用效率和便利性。
Hiddify Next 如何设置用户带宽限制？
管理员可为每个用户设置独立带宽和流量配额，超出时可自动降速或阻止访问。这样可确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络效率和公平性。
SSR 是否还值得使用？
虽然 SSR 的开发早已停止，但由于它配置简单且部分节点仍然兼容，仍有一定使用场景。不过建议用户逐步迁移到更安全的新协议如 VLESS 或 Trojan，以获得更佳性能与隐蔽性。
Sing-box 如何实现精确域名分流？
Sing-box 可以按域名设置分流规则，将不同流量发送到指定节点。结合策略组使用，可实现网页、视频和游戏流量精准分配，提高访问速度和网络稳定性。
